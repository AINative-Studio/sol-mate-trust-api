## AUTHENTICATION
Flow: Wallet signature challenge
1. POST /api/v1/auth/wallet/connect { "wallet_address": "..." } → challenge string
2. Sign challenge with Solana wallet
3. POST /api/v1/auth/wallet/connect { "wallet_address", "signature", "challenge" } → JWT
4. Use: Authorization: Bearer <token>

## ACTIONS
- Discover rooms:          GET  /api/v1/rooms
- Create stake:            POST /api/v1/stakes
- Request match:           POST /api/v1/matches
- Send message:            POST /api/v1/messages
- Submit attestation:      POST /api/v1/attestations
- Gift SOL:                POST /api/v1/transfers
- Mint Moment NFT:         POST /api/v1/nfts/mint-moment
- List Moment NFTs:        GET  /api/v1/nfts/moments

## STAKE MECHANICS
- DM stake: 0.01 USDC minimum
- Room stake: 0.10 USDC minimum
- Meetup stake: 1.00 USDC minimum
- No-show: 50% slash base, escalating 0.5x per offense, 3x cap
- Harassment/fraud: 100% slash, always
